# Designer.md

## 1. Objective
To act as a guiding specification for an AI agent to generate a comprehensive `Design.md` document. The system will analyze an existing software program's design, integrate newly requested design specifications, and output a structured `Design.md` file while strictly preserving the core identity and foundational structure of the original product.

## 2. Tools
* **Main Engine:** Antigravity
* **AI Assistant:** Any AI agent

## 3. Scope
Applicable to any software project.

## 4. Inputs
The following fields must be provided to the AI agent for each design iteration:
* **Original Project Reference:** [Repository link, existing design file, or source code of the current program]
* **New Design Requests:** [Specific UI/UX enhancements, structural design changes, or new visual themes to be integrated]

## 5. Configuration
*(Note: All predefined standards have been reset. Any selections not explicitly detailed are strictly assigned as "Custom" to dynamically fit the target project without relying on outside assumptions.)*

* **Design System / UI Framework:** Custom (To be determined by the original project)
* **Styling Language / CSS Strategy:** Custom 
* **Component Architecture:** Custom
* **Color Palette / Typography:** Custom

## 6. Execution Directives
The AI agent must strictly follow these steps in order:

1.  **Analysis & Extraction:** Analyze the Original Project Reference. Identify the current design architecture, layout mechanics, and core visual components. **Strictly preserve** the original product's integrity and do not remove its primary functionality.
2.  **Synthesis:** Integrate the New Design Requests into the existing framework. Ensure that the new requirements enhance the current design without breaking the original structural logic.
3.  **Output Generation:** Generate a detailed `Design.md` file. This new file must clearly document the updated design guidelines, component states, layouts, and stylistic rules resulting from the combination of the original project and the new requests.
