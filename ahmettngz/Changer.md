# Changer.md

## 1. Objective
The objective is to clone an existing open-source software product (based on a referenced repository or link), rebuild it with requested new extensions, and localize the project. The system will integrate new features and topics while strictly preserving the core structure and integrity of the original product.

## 2. Tools
* **Main Engine:** Antigravity
* **AI Assistant:** Codex / Any designated AI tool

## 3. Scope
Mobile or PC applications, page agent-style extensions, and all accessible open-source software products.

## 4. Inputs
The following fields must be filled for each new iteration of the project:
* **Original Program Name:** [Name of the program to be cloned]
* **Reference Link:** [GitHub repository or source link]
* **New Topics/Features to Add:** [New functions and localization requirements to be integrated into the program]

## 5. Configuration
*(Note: Karpathy's program.md standards have been reset. Any selections not found or explicitly specified in the reference document are left as "Custom" to adapt dynamically to the target project.)*

* **Language:** Custom (Will adapt to the original repository's language)
* **Framework/Library:** Custom (Based on the original structure's requirements)
* **Environment:** Custom (Based on the target local environment)
* **Architecture:** Custom

## 6. Execution Directives
The AI tool must strictly execute the following steps in order:

1.  **Cloning and Analysis:** Analyze the original project from the provided link. **Strictly preserve** the operational logic and core components of the original product.
2.  **Localization:** Create configurations to adapt the project to the user's targeted local working environment.
3.  **Extension:** Develop the new topics and features specified in the Inputs section. Do not disrupt or break the working principles of the original code while implementing these enhancements.
4.  **Output:** Generate the new file structure, updated code blocks, and installation steps for the transformed project in accordance with this document.
